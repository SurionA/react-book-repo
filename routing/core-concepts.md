# Routing

## Type of routers

- BrowserRouter
- HashRouter


## Install and set up
Installing the router

```
npm install --save react-router-dom
```

### Adding the router

```js
import { BrowserRouter } from 'react-router-dom'

ReactDOM.render((
  <BrowserRouter>
    <App />
  </BrowserRouter>
), document.getElementById('root'))
```
